# 硅基流动免费模型清单

> 数据采集时间：2026-07-03
> 数据来源：`https://siliconflow.cn/pricing` + `https://cloud.siliconflow.cn/me/models`（登录后真实数据）

## 数据获取方式

1. **定价页** (`siliconflow.cn/pricing`)：使用 MCP Chrome DevTools 浏览器打开，点击 3 个「展开更多」折叠按钮（共展开 1 + 6 + 30 = 37 个被折叠模型），共扫描到 **87 个模型**
2. **模型广场** (`cloud.siliconflow.cn/me/models`)：登录后逐个导航至每个免费模型详情页（抽屉面板），提取官方标注的「支持能力」「上下文」「发布日期」与完整介绍
3. **重要发现**：
   - 静态抓取（如 WebFetch）会漏掉 **6 个被折叠的免费模型**，必须用浏览器展开折叠区才能拿到
   - 定价页将以下 2 个模型标为「免费」，但模型广场真实标注为「**限免**」（限时免费），已从免费清单中剔除：`deepseek-ai/DeepSeek-OCR`、`tencent/Hunyuan-MT-7B`
4. 最终确认 **14 个永久免费模型**

## 「支持能力」定义说明

硅基流动在模型广场详情页为每个模型标注「支持能力」，使用图标标识：

- 🛠️ 工具调用（Function Calling）
- 👁️ 视觉输入（多模态图像理解）
- ✍🏻 前缀续写（Prefix Completion）

嵌入 / 重排序 / 生图 / 语音模型不标注「支持能力」字段（非对话类模型）。

### 「思考模式」分类说明（本文档补充，非硅基流动官方字段）

> 硅基流动官方能力图标**不包含「思考模式」分类**。下文对话模型表格的「思考模式」列由本文档根据模型官方介绍推断，用于补充项目 `server/config/models.ts` 中 `deepThinking` 能力的划分依据。四档定义：

- **强制**：纯推理模型，API 不接受 `enable_thinking` 参数（GLM-Z1 传了会 400 报错，R1 传了被忽略），永远输出思维链（如 R1 蒸馏、GLM-Z1 系列）
- **默认启用**：支持双模式，默认走思考模式，可通过 `enable_thinking=false` 关闭（如 Qwen3.5-4B、Qwen3-8B）
- **不支持**：无思考模式（如 Qwen2.5、GLM-4 非 Z1 系列、OCR 专用模型）
